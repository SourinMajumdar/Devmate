Devmate is a modern developer dashboard to showcase projects, track activity, and visualize growth - all in one place.

## Features
- Developer profile header
- Project management (add, edit, delete)
- Analytics snapshot with growth chart
- Content timeline for recent activity
- Dark-themed, dashboard-style UI

## Tech Stack
- React + Vite
- CSS (custom styles)
- Kombai (UI acceleration)

## Getting Started

```bash
npm install
npm run dev
```
